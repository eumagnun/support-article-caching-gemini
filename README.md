Context Caching com Gemini no Vertex AI

Este repositório contém o código de exemplo e os scripts de suporte para o Artigo: Economizando Tokens do Gemini  através do uso de Context Caching Implícito e Explícito.

O objetivo deste projeto é demonstrar, de forma prática, como implementar as estratégias de Cache Implícito e Cache Explícito utilizando a API REST do Vertex AI para otimizar a análise de documentos extensos, como processos judiciais e administrativos.
